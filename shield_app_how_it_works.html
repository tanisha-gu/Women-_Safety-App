SHIELD Women Safety App — How It Works

🛡️ SHIELD — Women Safety App

AI-powered real-time threat detection, emergency escalation & police integration

What happens when danger is detected

📡
Trigger detected
Voice, shake, button, or AI audio analysis
🚨
SOS activated
GPS captured, recording starts, alert sent
📲
Contacts notified
SMS + push with live location link
👮
Police alerted
Control room receives live feed & coordinates
🔒
Evidence saved
Audio/video uploaded to cloud, geotagged

Three ways to trigger SOS

🎙️
Voice trigger
Say "Help me" — hands-free, no screen interaction needed
📳
Shake trigger
Shake phone 3× rapidly — works silently in pocket
🤖
AI auto-detect
Listens for screams (3–4kHz), triggers automatically

Technical stack

Frontend
Progressive Web App
  • Single HTML file
  • Offline-capable (PWA)
  • Mobile-first UI
  • Service Worker cache
  • Works on 2G networks
Backend
Node.js + Socket.IO
  • Express REST API
  • JWT authentication
  • WebSocket real-time
  • Rate limiting (100/15m)
  • MongoDB database
Integrations
External services
  • Twilio SMS alerts
  • AWS S3 evidence
  • Google Maps API
  • Police portal API
  • Push notifications

AI modules

🔊
Audio threat detection
Amplitude + frequency analysis (3–4kHz = scream range), confidence 0–100
🧠
Safe route scoring
Lighting + crowd + CCTV + police presence + incident history
⚠️
Risk assessment
Time-of-day weighting (night +35pts), area classification, ML scoring
